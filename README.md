## Installation
```
yarn
```
or
```
npm install
```

## Development
```
yarn dev
```
or
```
npm run dev
```

## Testing & Coverage
```
yarn test
```
or
```
yarn run test
```

## Build & Deployment
```
yarn build
yarn deploy
```
or
```
npm run build
npm run deploy
```
